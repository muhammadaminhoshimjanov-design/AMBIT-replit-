The onboarding must introduce the app gradually, personalize the experience, and set up the user profile in a way that increases attachment from the first minute.

- Premium
- Modern
- Smooth
- Slightly elite
- Student-focused
- Ambitious but not arrogant
- Warm and motivating
- High-end startup feel
- Not childish
- Not generic education-app style

CORE VISUAL IDENTITY:
App name: Ambit
Mascot: a friendly, confident lion wearing a cap/hat with the Ambit logo on it
Mascot role: guide the student through onboarding step by step
The lion should feel smart, motivating, and welcoming — like a mentor/guide, not like a cartoon clown

- Smooth card-based layouts
